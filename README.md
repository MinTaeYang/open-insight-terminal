# 🌐 OPEN INSIGHT TERMINAL

전 세계 실시간 뉴스 데이터를 무제한으로 조회하고 분석하는 하이테크 터미널입니다.

## 🚀 주요 기능
- **실시간 뉴스 분석**: Google News RSS Feed를 통한 최신 데이터 수집
- **하이테크 UI**: 다크 모드 기반의 가독성 높은 대시보드
- **키워드 검색**: 마켓, 자산, 기업 등 자유로운 키워드 분석

## 🛠 기술 스택
- **Language**: Python
- **Framework**: Streamlit
- **Library**: feedparser, urllib

## 💻 설치 및 실행
1. 저장소 클론
2. 라이브러리 설치: `pip install -r requirements.txt`
3. 앱 실행: `streamlit run app.py`
